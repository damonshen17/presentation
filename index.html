The Power of color-mix()

Modern CSS Color Manipulation

The "Traditional" Way

Let's start with one button and see how the code grows...

            
              /* Red Button */
              .btn-red {
                background-color: #e53e3e;
                color: white;
                border: 2px solid #ff6b6b;
              }
              .btn-red:hover {
                background-color: #c53030;
                border-color: #ff4444;
              }

The "Traditional" Way

But what about more colors? Now we have to manage 6 different values.

            
              /* Red Button */
              .btn-red {
                background-color: #e53e3e;
                color: white;
                border: 2px solid #ff6b6b;
              }
              .btn-red:hover {
                background-color: #c53030;
                border-color: #ff4444;
              }

              /* Blue Button */
              .btn-blue {
                background-color: #3182ce;
                color: white;
                border: 2px solid #6bb6ff;
              }
              .btn-blue:hover {
                background-color: #2b6cb0;
                border-color: #4499ee;
              }

              /* Green Button */
              .btn-green {
                background-color: #38a169;
                color: white;
                border: 2px solid #6bffa1;
              }
              .btn-green:hover {
                background-color: #2f855a;
                border-color: #44ee88;
              }

A "Smarter" Way: CSS Filters

We can refactor to a generic .btn class with a hover filter...

            
              .btn {
                color: white;
                transition: filter 0.2s;
              }
              .btn-red {
                background-color: #e53e3e;
                border: 2px solid #ff6b6b;
              }
              .btn:hover {
                filter: brightness(85%);
              }

A "Smarter" Way: CSS Filters

This is better! Now we only need to define the base color for each button.

            
              .btn {
                color: white;
                transition: filter 0.2s;
              }
              .btn-red {
                background-color: #e53e3e;
                border: 2px solid #ff6b6b;
              }
              .btn:hover {
                filter: brightness(85%);
              }
              .btn-blue {
                background-color: #3182ce;
                border: 2px solid #6bb6ff;
              }
              .btn-green {
                background-color: #38a169;
                border: 2px solid #6bffa1;
              }

The Modern Way: color-mix()

Using color-mix() gives us the most power and control...

            
              .btn {
                background-color: var(--base-color);
                color: white;
                border: 2px solid color-mix(in oklch, var(--base-color), white 30%);
                transition: background-color 0.2s;
              }
              .btn:hover {
                background-color: color-mix(in oklch, var(--base-color), black 20%);
              }
              .btn-red {
                --base-color: oklch(63% 0.26 29);
              }

The Modern Way: color-mix()

This is the ultimate solution. Just define one --base-color per button.

            
              .btn {
                background-color: var(--base-color);
                color: white;
                border: 2px solid color-mix(in oklch, var(--base-color), white 30%);
                transition: background-color 0.2s;
              }
              .btn:hover {
                background-color: color-mix(in oklch, var(--base-color), black 20%);
              }
              .btn-red {
                --base-color: oklch(63% 0.26 29);
              }
              .btn-blue {
                --base-color: oklch(60% 0.18 264);
              }
              .btn-green {
                --base-color: oklch(65% 0.19 150);
              }
